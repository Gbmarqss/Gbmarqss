# Olá, eu sou o Gabriel Marques! 👋

### 💻 Desenvolvedor Backend & Infraestrutura

Focado no desenvolvimento de aplicações de alta performance e na criação de infraestruturas resilientes. Atuo na **convergência entre código e hardware**, otimizando recursos desde o gerenciamento de memória em aplicações até a administração de sistemas operacionais.

🔭 **Foco Técnico Atual:**
- **Desenvolvimento de Software:** Criação de aplicações Desktop de alto desempenho processando arquivos complexos (RAW) sem bloqueio de UI usando **Rust + Tauri**.
- **Infraestrutura e DevOps:** Containerização de serviços, automação de deploys com **Docker** e **Shell Script**, e integração contínua (CI/CD).
- **Sistemas Operacionais:** Administração avançada e otimização de ambientes **Linux (Debian/Pop!_OS)**.
