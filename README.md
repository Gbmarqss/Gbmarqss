### 🎓 Desenvolvedor Backend & Infraestrutura em Formação

Sou um estudante de Análise e Desenvolvimento de Sistemas apaixonado pela **convergência entre código e hardware**. Meu foco é desenvolver aplicações de alta performance e criar infraestruturas resilientes.

Não me contento apenas em fazer o código funcionar; gosto de entender o gerenciamento de memória, o sistema de arquivos e como otimizar recursos em ambientes Linux.

🔭 **No momento, estou trabalhando em:**
- 📸 **ChurchFlow:** Uma aplicação Desktop de alta performance em **Rust + Tauri** para processamento de imagens RAW.
- 🐳 Containerização de aplicações e automação de ambientes com **Docker** e **Shell Script**.
- 🐧 Aprofundamento em administração de sistemas **Linux (Debian/Pop!_OS)**.

🌱 **Estudando atualmente:**
- 🦀 **Rust:** Gerenciamento de memória e Concorrência (Threads).
- 🏗️ **DevOps:** CI/CD e Infraestrutura Ágil.
- 🧠 **IA Aplicada:** Integração de LLMs para otimização de código e fluxos de trabalho.
